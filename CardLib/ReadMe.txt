CardLib
A Win32 C++ card-game library.

Freeware
Copyright J Brown 2001
------------------------------

Unzip all these files into your source directory.
There are two directories of files, which should
both be "side-by-side". e.g.

c:\src\CardLib
c:\src\CardExample

CardLib contains the library files. Include the CardLib.dsp
project, or CardLib.lib in your own project.

CardExample is a simple example of how to use the library.
